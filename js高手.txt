js中的一些特殊运算符
1、~~可以实现向下取整，先将这个数取绝对值，然后向下取整，在加上原来的符号如
也可以认为是一种强行转换位number整数类型console.log(typeof ~~'2') // number
console.log(~~3.7) //3 console.log(~~ -3.2)//-3 console.log(~~'2.1') // 2
2、<< 向左移位 如：14 << 2 将14转化为2进制然后向左移2位；向左移2位相当于增大4倍，向左移几位就是乘以2的多少次方 
3、void 0 === undefined; //true
解释 void 0 的意思是将0强制性转化为无类型
4、255转为2进制是8个1，这样的话就可以实现rgb中颜色值和255用&用算时都是原来的值
200 & 255 // 200 17 & 255 // 17
如果是2的倍数则它转位2进制之后是1000...
